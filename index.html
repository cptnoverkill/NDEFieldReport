Generate PDF Report

The PDF generator runs locally via Python and produces a formatted report matching the Acuren UTT report style.

Step 1 — Export your field data
Click the button below to download the JSON file
Step 2 — Install requirements (one time only)
pip install reportlab Pillow
Step 3 — Run the PDF generator
Place generate_pdf.py in the same folder as the JSON, then run: python generate_pdf.py field_data.json
Step 4 — Open the output PDF
The PDF will be saved in the same folder as the JSON file.
